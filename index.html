About TW5FontAwesome

TW5FontAwesome supports embedding  Font Awesome in  TiddlyWiki 5. There is no need to install this font into your operating system in order to use it with TiddlyWiki 5. The font is already embedded in this TiddlyWiki 5 customization instead, so nothing to install at all. Please see the demo (link below) for details on how to easily install Font Awesome support into your existing TiddlyWiki 5 instances. It's basically a simple drag&drop operation.
